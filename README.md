# Movie/Actor Guesser – Portfolio Project
## Description

Ce projet est une application Node.js qui permet de deviner des films et des acteurs à travers un graph interactif, avec plusieurs fonctionnalités fun et dynamiques :

- Les films et acteurs sont tirés aléatoirement pour chaque round.
- Timer intégré : Chaque question est chronométrée pour plus de challenge.
- Système de score
- Génération de memes aléatoires : Utilise une clé API pour créer des memes en lien avec le jeu.
- Graph interactif : Explore les relations entre acteurs et films pour trouver les bonnes réponses.

Ce projet a été conçu comme un portfolio pour montrer mes compétences en Node.js, gestion de bases de données, et intégration d’APIs.

---

## Installation

### Cloner le projet :
`
git clone https://github.com/AledTahLesFous/sae_402.git
cd movie_actors_api
`

### Installer les dépendances :

`
npm install
`

### Lancer le serveur :

`
npm start
`

Technologies utilisées

- Node.js
- Sequelize ORM
- SQLite (base de données locale pour simplifier le déploiement)
- API externe pour la génération de memes
- Graph JS pour visualiser les relations films/acteurs

## Usage

Accéder au jeu depuis http://localhost:3000
